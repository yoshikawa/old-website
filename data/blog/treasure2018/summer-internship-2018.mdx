### Treasure に参加した理由

きっかけは、サポーターズさんの 1 on 1 面談でした。

VOYAGE GROUP の人事の方に、「**Treasure は良いよ!絶対に後悔しないから!**」とめっちゃ言われました。

1 on 1 って案外疲れがたまるんですよね。何を聞き間違えたのか、「**インフラを触れるのが Treasure**」という謎の解釈があって Treasure に応募しました。

今となっては笑い話ですが、Treausre に参加したときに、講義についていけるか不安でした。

### 講義内容

#### 1 日目

Golang の講義がありました。

まずは、Treasure メンバーと馴染むために隣の人とペアプログラミングをしました。

TDD 開発を実感できる良い時間でした。

オリジナルの curl を作って 1 日目は終了しました。

詳しくは、[Treasure 1 日目](https://yoshikawa.dev/blog/treasure2018/day1/)

#### 2 日目

Golang の講義がありました。

TODO アプリを拡張しました。

SQL と Golang を繋げる作業。SQL を叩いて Golang でパースすることが出来ました。

TODO アプリの API を作成し、curl コマンドで TODO の検索を行いました。

詳しくは、[Treasure 2 日目](https://yoshikawa.dev/blog/treasure2018/day2/)

#### 3 日目

React の講義がありました。

Flux, Redux などを触った。

redux-thunk に触れた。

React で TODO アプリを作成した。

詳しくは、[Treasure 3 日目](https://yoshikawa.dev/blog/treasure2018/day3/)

#### 4 日目

WebAPI の講義がありました。

API 設計はどうすればいいかを学んだ。

良いものをコピーしよう!

- GitHub
- AWS S3
- PayPal

インターン中にバズった内容は、この WebAPI の講義でした。

#### 7 日目

データベースの講義がありました。

モデリングの手法を学んだ。 ER 図の書き方を学びました。

3 人 1 組で、ポイントサイトを作成する際の DB 設計について話し合いをしました。

詳しくは、[Treasure 7 日目](https://yoshikawa.dev/blog/treasure2018/day7/)

#### 8 日目

チーム開発の講義がありました。

チームとはそもそも何かから始まり、どういう開発手法があるかを教えていただきました。

残りの時間で、チームに分かれて相手が求めた街を資材(折り紙など)で作っていく模擬をしました。

相手が何を求めているか、フィードバックをして次回に活かす開発手法は僕に合っていると感じました。

最後に、後半の 1 週間半でアプリ開発をするためのチーム発表がありました。

詳しくは、[Treasure 8 日目](https://yoshikawa.dev/blog/treasure2018/day8/)

アイデア出しの講義がありました。

時流(時の流れ)を掴むのが大事ということを学びました。

例えば、ストリーミングサービスの普及の裏には、個人の所有というものが薄くなったから とか。

詳しくは、[Treasure 9 日目](https://yoshikawa.dev/blog/treasure2018/day9/)

### 参加した学生の質の良さ

みんな、プログラミングができる学生さんでした。

ただ、それだけじゃないんですよ。なんというか人間性と言いますか、本当に良い奴らでした。

僕がふざけたことをふざけて返してくれたり、真面目に考えてるときには手を差し伸べてくれたり。

最高のメンバーでした。

### 僕たちが作ったアプリ

僕らのような学生エンジニアを対象とした、エンジニアインターンのリアルを伝えるサイト **「Baton Road」** を作成しました。

![baton road](/static/images/treasure/baton-road.png)

ログインは Facebook OAuth 認証を用いています。

インターンの単位を本として定義し、その中に記事が格納されています。

チームが 6 チームあったのですが、僕らのチームは 3 位という結果で終わりました。

ですが、僕の中では Quiet404 は 1 位でした。

**これは、僕が保証します。**

![Quiet404](/static/images/treasure/quiet404.jpg)
